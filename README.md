# 👨🏻‍💻 Customer Behavior Analysis Project

A complete, end-to-end data analytics project designed to simulate real-world business analysis workflows — from raw data processing to executive-level insights and reporting.

---

## 📌 Project Overview

This project demonstrates a structured analytics pipeline:

* **Data Preparation & EDA (Python)** – Data cleaning, transformation, and exploratory analysis.
* **Business Analysis (SQL)** – Customer segmentation, loyalty analysis, and purchase behavior insights.
* **Visualization (Power BI)** – Interactive dashboard for stakeholder decision-making.
* **Reporting & Presentation** – Clear summary of insights and business recommendations.

---

## 🛠️ Tech Stack

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **SQL** (MySQL / PostgreSQL / MS SQL Server)
* **Power BI**
* **Jupyter Notebook**

## 🚀 How to Use

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/MrigenDeka04/Customer-Trend-Analysis.git
cd Customer-Trend-Analysis
```

### 2️⃣ Run the Python Notebook

Open `Customer_Shopping_Behavior_Analysis.ipynb` to:

* Import dataset
* Perform data cleaning
* Conduct exploratory analysis
* Connect to SQL database

### 3️⃣ Load Data from Python notebook into SQL

* Create a database (MySQL/PostgreSQL/MS SQL Server)
* Run Python script to load cleaned data
* Execute queries from `customer_behavior_sql_queries.sql`
* Answer Business Questions

### 4️⃣ Connect Power BI

* Open `customer_behavior_dashboard.pbix`
* Connect to your SQL database

### 5️⃣ Generate Report & Presentation

* Prepare a project report summarizing findings
* Build a presentation
